# Draw Me Dis!

A tiny doodle inspiration generator that works like a digital paint-night instructor.
Instead of one big drawing prompt, it reveals step-by-step instructions so you never feel overwhelmed.

---

## How It Works

1. Click **"Give me a doodle!"** — the app randomly picks an animal, an object, and a zentangle pattern and combines them into your unique doodle idea.
2. Click **"Next Step"** to reveal one drawing instruction at a time.
3. Follow each step at your own pace until your doodle is complete.
4. Type a name for your finished masterpiece and save it.
5. Click **"Start Over"** to generate a brand-new idea.

---

## How To Run

No install, no build step, no server needed.

Just open `index.html` in any modern web browser and start doodling.
